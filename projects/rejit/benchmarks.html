Benchmarks

Results automatically generated by Rejit benchmark tools. See the documentation for instructions on how to run the benchmarks.

Benchmark results in this page present performance to search for all (left-most longest) matches of a regular expression in a text.
Performance is reported in bytes per second: the size of the text matched divided by the time to process it. This can be interpreted as speeds at which files of given sizes can be processed.

The v8 results seem to go crazy in some situations. This needs to be investigated.
